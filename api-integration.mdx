## API Overview

Plyon's REST API allows you to programmatically manage documentation projects and content.

<Callout kind="info">All API endpoints require authentication using API keys.</Callout>

## Authentication

<Steps>
  <Step title="Get API Key" icon="key">
    Generate an API key from your account settings.
  </Step>
  <Step title="Include in Headers" icon="settings">
    Add the key to your request headers.
    
    ```bash
    curl -H "Authorization: Bearer YOUR_API_KEY" \
         https://api.plyon.com/v1/projects
    ```
  </Step>
</Steps>

## Core Endpoints

<Tabs>
  <Tab title="Projects" icon="folder">
    Manage documentation projects.
    
    <Request tabs="Create Project,List Projects">
      ```json
      {
        "name": "My Project",
        "description": "API documentation"
      }
      ```
      ```json
      [
        {
          "id": "proj_123",
          "name": "My Project"
        }
      ]
      ```
    </Request>
  </Tab>
  <Tab title="Documents" icon="file-text">
    Create and update documents.
    
    <Response tabs="Document Response">
      ```json
      {
        "id": "doc_456",
        "title": "API Guide",
        "content": "# API Guide\n\nThis is..."
      }
      ```
    </Response>
  </Tab>
</Tabs>

## Error Handling

Handle API errors gracefully.

<Callout kind="alert">Check response status codes and error messages for debugging.</Callout>

| Status Code | Meaning |
|-------------|---------|
| 200 | Success |
| 400 | Bad request |
| 401 | Unauthorized |
| 404 | Not found |
| 500 | Server error |
